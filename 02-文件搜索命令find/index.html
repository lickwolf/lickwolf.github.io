    文件搜索命令¶

    命令名称：find

    命令所在路径：/bin/find

    执行权限：所有用户

    语法：find [搜索范围] [匹配条件]

    功能描述：文件搜索

    //不要在服务器使用高峰期时候使用find进行查找，会占用大量资源

    1.精确搜索

    2.使用通配符的模糊搜索

    3.使用带有？的搜索

    4.不区分大小写的搜索

    5.根据文件大小进行查找

    locate

    命令所在路径：/usr/bin/locate

    执行权限：所有用户

    语法：locate 文件名

    功能描述：在文件资料库中查找文件

    eq：

    1.查找inittab

    [root@centos6 ~]# locate  inittab
/etc/inittab
/usr/share/augeas/lenses/dist/inittab.aug
/usr/share/man/man5/inittab.5.gz
/usr/share/vim/vim74/syntax/inittab.vim

    2.在/root下创建文件miaomiao；紧接着使用locate命令无法查找；需要使用updatadb更新数据库之后进行查找

    [root@centos6 ~]# touch /root/miaomiao
[root@centos6 ~]# locate miaomiao
[root@centos6 ~]# updatedb
[root@centos6 ~]# locate miaomiao
/root/miaomiao

    3.临时文件夹tmp 不适合updatadb

    [root@centos7 ~]# touch /tmp/zhangbozhi
[root@centos7 ~]# updatedb
[root@centos7 ~]# locate zhangbozhi

    4.不严格区分大小写查找

    [root@centos7 ~]# locate -i miaomiao
/home/miaomiao
/home/miaomiao/.bash_history
/home/miaomiao/.bash_logout
/home/miaomiao/.bash_profile
/home/miaomiao/.bashrc
/home/miaomiao/.cache
/home/miaomiao/.config
/home/miaomiao/.mozilla
/home/miaomiao/.cache/abrt
/home/miaomiao/.cache/abrt/lastnotification
/home/miaomiao/.config/abrt
/home/miaomiao/.mozilla/extensions
/home/miaomiao/.mozilla/plugins
/root/Miaomiao
/var/spool/mail/miaomiao

    which

    命令所在路径：/usr/bin/which

    执行权限：所有用户

    语法：which 命令

    功能：搜索命令所在的目录及别名信息

    eq：

    1.

    [root@centos7 ~]# which ls
alias ls='ls --color=auto'
        /usr/bin/ls
[root@centos7 ~]# which rm
alias rm='rm -i'
        /usr/bin/rm

    whereis

    功能：搜索命令所在的目录及帮助文档路径

    eq：

    [root@centos7 ~]# whereis ls
ls: /usr/bin/ls /usr/share/man/man1/ls.1.gz /usr/share/man/man1p/ls.1p.gz

    grep

    命令所在路径：/bin/grep

    执行权限：所有用户

    语法： grep -iv [制定字符串] [文件]

    功能：在文件中搜寻字符串匹配的行并输出

    ​ -i 不区分大小写

    ​ -v 排除制定字符串

    ​ -n 显示行号

    eq：

    1.查找/etc/inittab 中的systemctl所在的行

    [root@centos7 ~]# grep systemctl /etc/inittab
# systemctl get-default
# systemctl set-default TARGET.target

    2.查找target不区分大小写

    [root@centos7 ~]# grep -i target /etc/inittab
# Ctrl-Alt-Delete is handled by /usr/lib/systemd/system/ctrl-alt-del.target
# systemd uses 'targets' instead of runlevels. By default, there are two main targets:
# multi-user.target: analogous to runlevel 3
# graphical.target: analogous to runlevel 5
# To view current default target, run:
# To set a default target, run:
# systemctl set-default TARGET.target

    3.将开头为#的注释行省略//^符号在前

    [root@centos7 ~]# grep -v ^# /etc/inittab
www.hcieonlin.com

    4.查找target在/etc/inittab中；要求，省略显示#开头，查找结果显示行号

    [root@centos7 ~]# grep -nv ^#  target /etc/inittab
grep: target: No such file or directory
/etc/inittab:18:www.hcieonlin.com
