    命令格式与目录处理命令ls¶

    命令格式：命令 [-选项] [参数]

    选项：用来调整命令功能的

    参数：指定命令操作的对象

    例如： ls -la /etc

    说明：

    ​ 1.个别命令不遵循此格式

    ​ 2.当有多个选项时，可以写在一起

    ​ 3.简化选项与‘’完整选项

    ​ -a 等于 --all：能够显示出所有文件，包括隐藏文件

    目录处理命令¶

    mkdir

    命令名称：mkdir

    NAME mkdir - make directories

    创建目录

    [root@centos7 ~]# mkdir  /tmp/boduo

    创建多个目录

    [root@centos7 ~]# mkdir /tmp/cangjing /tmp/longze

    在多个位置同时创建目录

    [root@centos7 ~]# mkdir /tmp/cangjing/nvyou /tmp/taogu

    目录切换命令：cd

    英文原意：change directory

    显示当前所在目录：pwd

    pwd - print name of current/working directory//打印出所在的工作目录

    命令所在路径：shell内置命令

    执行权限：所有用户

    语法：cd [目录]

    功能描述：切换目录

    范例：

    1.切换到 /tmp 目录下；[root@centos7 tmp]# [] 内显示的自己所在目录

    [root@centos7 ~]# cd /tmp
[root@centos7 tmp]#

    2.切换多级目录

    [root@centos7 ~]# cd /tmp/cangjing/nvyou/
[root@centos7 nvyou]#

    3.返回上一级目录 cd ..

    .. 表示上一级目录

    . 表示本目录

    [root@centos7 nvyou]# cd ..
[root@centos7 cangjing]#

    删除空目录：rmdir

    NAME rmdir - remove empty directories

    语法：

    rmdir [OPTION]... DIRECTORY...

    例子：

    1.删除空目录 taogu

    [root@centos7 tmp]# rmdir taogu
[root@centos7 tmp]#

    2.尝试删除非空目录

    //删除失败，因为该目录非空

    [root@centos7 tmp]# rmdir cangjing/
rmdir: failed to remove ‘cangjing/’: Directory not empty

    复制命令：cp

    命令名称：cp

    命令英文原意：

    NAME cp - copy files and directories//复制文件或者目录

    格式：

    ​ cp [OPTION]... [-T] SOURCE DEST ​ cp [OPTION]... SOURCE... DIRECTORY ​ cp [OPTION]... -t DIRECTORY SOURCE...

    命令所在路径：/bin/cp

    执行权限：所有用户

    语法：

    cp -rp [源文件或目录] [目标文件]

    ​ -r 复制目录

    ​ -p 保留文件属性

    eq:

    1.复制目录到目录

    [root@centos7 tmp]# cp -r xiaodianying xiaodianying1
[root@centos7 tmp]# cd xiaodianying1
[root@centos7 xiaodianying1]# ls
xiaodianying

    2.复制文件到目录

    [root@centos7 ~]# cp readme.txt /tmp/xiaodianying
[root@centos7 tmp]# cd xiaodianying
[root@centos7 xiaodianying]# ls
readme.txt  xiaodianying

    3.保留源文件的属性并复制

    [root@centos7 ~]#  cp -p readme.txt  /tmp/xiaodianying
cp: overwrite ‘/tmp/xiaodianying/readme.txt’? y
[root@centos7 xiaodianying]# ls -l
^[[6~total 4

    删除命令：rm

    NAME rm - remove files or directories

    命令所在路径;/bin/rm

    -r, -R, --recursive remove directories and their contents recursively

    -f, --force ignore nonexistent files and arguments, never prompt

    eq:

    1.删除目录//有询问

    [root@centos7 ~]# rm -r xiaodianying/

    2.强制删除目录//不执行询问

    [root@centos7 ~]# rm -rf xiaodianying/

    链接命令¶

    命令英文原意：link

    命令所在路径:/bin/ln

    执行权限：所有用户

    语法：ln -[s] [源文件] [目标文件] //有s为软连接 无s为硬链接

    ​ -s 创建软连接

    功能描述生成链接文件

    eq:

    1.创建软连接文件

    ln -s /etc/issue /tmp/issue.soft

    显示软连接文件

    lrwxrwxrwx. 1 root root    10 Oct 25 23:42 issue.soft -> /etc/issue

    软链接文件的特征：

    lrwxrwxrwx l：表示连接 rwxrwxrwx：所有用户都有读写操作的权限

    类似于windows的快捷方式

    软连接的权限并不决定源文件的权限

    软连接文件的大小非常小，因为仅仅是一个符号链接 箭头指向源文件

    2.创建硬链接

    硬链接的特征：

    1.拷贝cp-p+同步更新

    echo“www.hcieonline.con” >> /etc/issue

    [root@centos7 tmp]# less issue.hard

\S
Kernel \r on an \m

www.hcieonline.com
issue.hard (END)
